WRITERS ROOM — SETUP GUIDE

Deploy your Writers Room

Five steps from empty folder to live at writersroom.fredericlabadie.com

01
Repo & Local Setup
Create the GitHub repo, install dependencies, set up your local environment file.
~10 min
→
02
Supabase Setup
Create the Postgres database, run the schema (profiles, rooms, members, messages), get API keys.
~10 min
→
03
OAuth Setup
Create Google Cloud and GitHub OAuth apps. Configure callback URLs. Get your Anthropic API key.
~15 min
→
04
Vercel Deploy
Import repo to Vercel, add all environment variables, deploy. Set up your custom domain in Vercel settings.
~10 min
→
05
Scaleway DNS
Add a CNAME record in your Scaleway DNS zone pointing writersroom → Vercel. Wait for propagation.
~5 min + propagation
→

After you're live

Every git push to main auto-deploys to Vercel. To add collaborators, share the invite code shown in any room's header. Private rooms are invite-only; public rooms appear to all logged-in users.

To add a new agent, add an entry to lib/personas.ts — no other changes needed. The UI and API both read from that single source of truth.
